CSS (Cascading Style Sheets)

CSS es un lenguaje de diseño usado para describir la presentación de un documento HTML. Permite controlar el diseño, los colores, fuentes y otros aspectos visuales de un sitio web.

JavaScript

JavaScript es un lenguaje de programación que permite implementar funciones complejas en páginas web. Hace que las páginas web sean interactivas y dinámicas.
